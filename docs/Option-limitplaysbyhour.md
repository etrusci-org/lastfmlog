# [LastfmLog](../README.md) - Option: `--limitplaysbyhour`

`--limitplaysbyhour NUMBER`

Applies to action: `stats`

Limit the number of items in plays by hour.


---


## Example

```text
cli.py stats --limitplaysbyhour 10
```
