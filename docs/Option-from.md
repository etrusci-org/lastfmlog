# [LastfmLog](../README.md) - Option: `--from`

`--from UNIXTIME`

Applies to action: `update`

Only fetch plays after this time. Unixtime stamp must be in UTC.


---


## Example

```text
cli.py update --from 1685611825
```

```text
downloading page:1 from:1685611825 to:latest limit:20
saved 8 tracks
```
